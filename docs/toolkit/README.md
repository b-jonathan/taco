# Toolkit

This folder documents small, reusable helper packages used by taco: command execution, filesystem helpers, GitHub and git wrappers, Node package helpers, and prompt utilities.

Use these pages when you need to understand or extend low-level helpers used across stacks and the CLI.

Pages in this folder:
- `exec.md` — running external commands (`internal/execx`)
- `fs.md` — filesystem helpers and templates (`internal/fsutil`)
- `gh.md` — GitHub client wrapper (`internal/gh`)
- `git.md` — git helper wrappers (`internal/git`)
- `nodepkg.md` — package.json helper (`internal/nodepkg`)
- `prompt.md` — survey/prompt helpers (`internal/prompt`)

See also: `docs/architecture/helpers.md` for a short overview.
